# Heart Disease Risk Analysis 🩺

End-to-end Data Science project on the **UCI Heart Disease dataset**

**[Click here to try the Live Prediction Dashboard](https://heart-disease-analysis-9j7uujtrac6vyjjc5byajf.streamlit.app)**

---

### 📊 Dataset
- **Source**: UCI Heart Disease Dataset (via Kaggle)
- **Size**: 920 patients, 16 features

---

### 🔍 Day 1 — Exploratory Data Analysis

**Key findings from the data:**

- 44.7% of patients have heart disease
- Males have a **63%** disease rate vs **25%** for females
- Lower maximum heart rate is the strongest predictor of disease
- Asymptomatic patients surprisingly have the highest disease count

Machine Learning (XGBoost)

**Model Performance:**

- **Accuracy**: 86.41%
- **ROC-AUC Score**: 0.9224
- **Top Feature**: Chest pain type (`cp`)
- Correctly identified **93 out of 102** disease cases
